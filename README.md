# Portfolio Website

A clean, professional portfolio website built with React and Bootstrap 5.

![Portfolio Screenshot](https://github.com/user-attachments/assets/9b023284-65c6-4a64-9c6f-23f500dba8cd)

## Features

- **Clean Design**: White and blue color scheme with red accents
- **Responsive**: Works on all device sizes
- **Bootstrap 5**: Modern UI components
- **Single Page**: Smooth scrolling navigation
- **Easy to Customize**: All data in one file

## Sections

- **Bio**: About me section with profile image
- **Projects**: Showcase your work with descriptions and links
- **Research**: Academic publications and papers
- **Interests**: Display your areas of interest
- **CV**: Education, experience, and skills
- **Updates**: Timeline of recent activities

## Getting Started

### Installation

```bash
npm install
```

### Development

```bash
npm start
```

Opens [http://localhost:3000](http://localhost:3000) in your browser.

### Production Build

```bash
npm run build
```

## Customization

Edit `src/data/portfolioData.js` to update:

- **personalInfo**: Your name, email, location, social links
- **bio**: Your biography text
- **projects**: Your project portfolio
- **research**: Your publications
- **interests**: Your areas of interest
- **updates**: Recent news and updates
- **cv**: Education, experience, and skills

## Color Scheme

The default colors can be customized in `src/index.css`:

- Primary Blue: `#0066cc`
- Accent Red: `#dc3545`
- Background: White/Light Gray

## Technologies

- React 19
- Bootstrap 5
