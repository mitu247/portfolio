# Portfolio Website

A clean, professional portfolio website built with React and Bootstrap 5.

## Tech Stack

- **React 18**: A JavaScript library for building user interfaces.
- **Bootstrap 5**: The world's most popular framework for building responsive, mobile-first sites.
- **FontAwesome**: For scalable vector icons.
- **CSS3**: Custom styling for a polished look.

## Use Cases

This project serves as a personal portfolio website to showcase:

- **Professional Profile**: A brief bio and introduction.
- **Skills**: A list of technical skills and expertise.
- **Experience**: Professional work history and education.
- **Projects**: A showcase of software development projects with descriptions and links.
- **Research**: Academic publications and research work.
- **Updates**: A timeline of recent career updates and achievements.
- **Contact**: Links to social profiles (LinkedIn, GitHub) and email.
